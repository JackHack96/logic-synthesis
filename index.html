UNIVR Logic Sythesis Software

Welcome to University of Verona's logic sythesis software web portal!
Here you can find some historical logic sythesis software, like SIS, which is still used for educational purposes at University of Verona.

Overview

This website hosts the following software:

  • The original SIS (version 1.3)
  • The refactored SIS (version 1.4)
  • MVSIS
  • Balm-II
They are all very old software, but they are still used in some courses, like Computer Architecture or Software for Embedded Systems at University of Verona. Reasons they're still used includes their relative simplicity and the closeness to theory, which is very useful for better understanding it.
