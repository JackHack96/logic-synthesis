UNIVR Logic Synthesis Software

Welcome to University of Verona's logic synthesis software web portal! The purpose of this site is to host source code, binaries and documentation of software for combinational and sequential logic synthesis. Currently, the following libraries are made available: Espresso, SIS, MVSIS and BALM. The software has been updated to make it compliant with current compilers. More specific information is available in the pages of each tool.
